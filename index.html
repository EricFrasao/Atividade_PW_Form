a) Na tag "input" com atributo type="text", para que serve o atributo "placeholder"?

Indica a função para o cliente, pré-visualização do que será respondido no local.

b) Na tag "input" com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

Checked é usado para quando o formulário for aberto a opção que houver esse atributo já aparecerá automaticamente marcada.

c) Na tag "input" para que serve o atributo "value"?

O atributo "value" é usado para específicar uma valor inicial no campo de entrada.

d) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

É utilizado o atributo "name", onde todos os valores devem ser iguais para que assim somente um de cada vez fique marcado.

e) Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

"Label" é a tag que serve para rotular os id's e auxilia pessoas que tem dificuldade de clicar em coisas muito pequenas, como caixas de seleções, pois é possível marcar por apenas selecionar a palavra.
O "for" indica o elemento específico para qual equivale no formulário.

f) Na tag "option" para que serve o atributo "selected"?

Atributo utilizado para mostra uma opção pré-selecionada assim que é aberto o formulário.

g) Nas tags de formulário para que serve o atributo "required"?

É um atributo que indica a obrigatoriedade de marcação ou de preenchimento de campo no formulário.

h) Na tag "form" para que serve o atributo "action"?

Atributo usado para informar para onde os dados do formulário serão enviados (Local no Servidor).

i) Na tag "form" para que serve o atributo "method"?

É o atributo que indica a forma que serão enviados os dados, podendo ser enviados como variáveis de URL, sendo eles method="get", ou HTTP com method="post".

j) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?

Para que o valor da senha digitada não apareça na barra de endereços devemos utilizar o valor "POST" no atributo "method", pois ele não permite que a senha apareça na URL.

k) Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?

O mais apropiado é utilizar o valor "POST", visto que ele não tem limitação de tamanho e é usado para enviar grande quantidade de dados.
